# 7. 上傳格式

依檔案大小選擇 FormData 或 Base64。

---

## 上傳格式

| 格式 | 說明 |
|---|---|
| **FormData** | 適合上傳檔案，支援多種資料類型。其他夾帶資料請後端包成一個 Object |
| **Base64** | 適合小型檔案，將檔案轉為字串格式 |

---

## FormData 使用方式

### 有包物件
```javascript
formData.set('req', new Blob([JSON.stringify(omit(modelValue, ['files']))], { type: 'application/json' }))
formData.set('file', modelValue.files[0])
```

### 沒包物件
```javascript
formData.set('newsSubject', model.value.newsSubject)
formData.set('newsContent', model.value.newsContent)
formData.set('newsStartDate', useFormatDate(model.value.newsStartDate))
```
