# 3. 日期與時間格式

統一字串格式，避免前後端轉換成本。

---

## 🔍 查詢時（後端 → 前端）

### 後端回傳格式
後端統一回傳 ISO 8601 格式，不可有小數點：

| 類型 | 回傳格式 |
|---|---|
| 年月日 | `1995-03-16` |
| 年月日時分 | `1995-03-16T18:14` |
| 年月日時分秒 | `1995-03-16T18:14:57` |

### 前端轉換顯示
前端依需求轉換為民國年：

| 類型 | 轉換結果 |
|---|---|
| 年月日 | `84/03/16` |
| 年月日時分 | `84/03/16 18:14` |
| 年月日時分秒 | `84/03/16 18:14:57` |

---

## 💾 儲存時（前端 → 後端）

### 前端送出格式
前端可送出以下任一格式：

**民國年格式：**

| 類型 | 格式 |
|---|---|
| 年月日 | `84/03/16` |
| 年月日時分 | `84/03/16 18:14` |
| 年月日時分秒 | `84/03/16 18:14:57` |

**或 ISO 格式（目前前端不儲存西元年，暫不啟用）：**

| 類型 | 格式 |
|---|---|
| 年月日 | `1995-03-16` |
| 年月日時分 | `1995-03-16T18:14` |
| 年月日時分秒 | `1995-03-16T18:14:57` |

### 後端接收規範
後端需能同時接受以下格式：

**民國年格式：**

| 類型 | 格式 |
|---|---|
| 年月日 | `84/03/16` |
| 年月日時分 | `84/03/16 18:14` |
| 年月日時分秒 | `84/03/16 18:14:57` |

**或 ISO 格式（目前後端會接收西元年，要能接這格式）：**

| 類型 | 格式 |
|---|---|
| 年月日 | `1995-03-16` |
| 年月日時分 | `1995-03-16T18:14` |
| 年月日時分秒 | `1995-03-16T18:14:57` |
